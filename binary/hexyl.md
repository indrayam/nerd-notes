# hexyl

## Installation

```bash
cargo install hexyl
```

## Usage

```bash
# Print version
hexyl -V

# Print help
hexyl -h

# Print a table showing how different types of bytes are colored
hexyl --print-color-table

# --base hexadecimal is the default
hexyl -b hexadecimal /path/to/file

hexyl /path/to/file
```
