# Warp Terminal

## Useful Shorcuts

```bash
# Terminal Window
Cmd + Shift + P # Command Pallete
Cmd + P # Navigation Pallete
Cmd + D # Split Pane Right
Cmd + Shift + D # Split Pane Down
Cmd + / # Keyboard Shortcuts
Cmd + K # Clear Blocks
Ctrl + L # Clear Screen

# Terminal Input Editor
Cmd + L # Focus Terminal Input
Cmd + Backspace # Delete All Left
Cmd + Delete # Delete All Right
Cmd + -> # Move to End
Cmd + <- # Move to Start
