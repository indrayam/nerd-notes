# spring cli

## Usage

### Get Help
```
spring help init
```

### Create Spring Boot project

```
spring init -a kotlin-app1 -g me.anandsharma -l kotlin -d=devtools,web,thymeleaf,jdbc,h2 --build=maven kotlin-app1
```
